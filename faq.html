Common Questions

What about the production environment?
The framework can already be used in a production environment. We have not provided tutorials (yet). That being said, be warned that this is a HIGHLY insecure environment. Do not expose it to the general public.
Is there a project Roadmap?
As of now, we are using GitHub Issue tracking. Feel free to submit anything you would like to see there.
Still have questions?

Check out our GitHub repository or open an issue for support.
