Our Mission

This project is intended to grow, scale, and tackle other versions of the Rails framework.

Additionally, we put a lot of love into making it seem like a more realistic application that can be enhanced to the point it mirrors a production application... only SUPER vulnerable. We hope you enjoy it, we hope you make it better.
